# Description
This repository contains code necessary to reproduce simulation results in "Transportability of Principal Causal Effects" 

For data privacy reasons, we are unable to share code that exactly reproduces our data analysis. 

We also include a script `estimate_nuisance_fcns_ml` that estimates nuisance functions using more flexible
models than those applied in the primary manuscript. This addition was recommended by a helpful suggestion during
the review process. We hope it might be useful for others applying our methods with alternate modeling strategies.
